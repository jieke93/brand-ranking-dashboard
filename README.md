# 유니클로 여성 랭킹 상품 정보 수집기

## ⚖️ 법적 고지사항

이 스크립트는 **교육 및 개인 연구 목적**으로 제작되었습니다.

### ✅ 허용 사항
- 개인 학습 목적의 데이터 수집
- 공개된 정보만 수집 (상품명, 가격, 이미지 URL, 컬러)
- robots.txt 규칙 준수
- 1회성 데이터 수집

### ❌ 금지 사항
- 상업적 사용
- 데이터 재배포 및 판매
- 대량 반복 크롤링
- 서버에 과부하를 주는 행위
- 개인정보 수집

### ⚠️ 법적 리스크 최소화 조치
- ✓ robots.txt 확인 및 준수
- ✓ User-Agent 명시 (봇 식별)
- ✓ 요청 간격 유지 (서버 부담 최소화)
- ✓ 공개 정보만 수집
- ✓ 저작권 표시 및 출처 명시

## 🔍 문제 해결

### Chrome 드라이버 오류
```bash
# webdriver-manager 재설치
pip uninstall webdriver-manager
pip install webdriver-manager
```

### 이미지 다운로드 실패
- 인터넷 연결 확인
- 방화벽/보안 프로그램 확인

### 상품을 찾을 수 없음
- 페이지 구조가 변경되었을 수 있음
- `page_source_debug.html` 파일 확인
- 스크립트 업데이트 필요

### robots.txt 차단 경고
- 차단된 경로는 수집하지 마세요
- 법적 문제가 발생할 수 있습니다
